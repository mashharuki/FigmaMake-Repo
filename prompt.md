```markdown
## Marvel's Spider-Man Remastered で提供されている機能

ご提示いただいたURLの『Marvel's Spider-Man Remastered』（PS5版およびPC版）で提供されている主な機能・特徴は以下の通りです。

このリマスター版は、PS4で発売されたオリジナル版をベースに、次世代機および最新PCの性能を最大限に引き出すための最適化と追加要素が施されています。

### 1. グラフィックスとパフォーマンスの向上

* **レイトレーシング:** 建物に映り込む反射やライティングがよりリアルに描写されます（対応デバイスが必要）。
* **パフォーマンスモード:** 60fps（フレーム秒間60コマ）での滑らかな動作を優先する設定が可能です。
* **ビジュアルの刷新:** キャラクターモデル、肌、目、髪の毛の質感が向上。さらに、主人公ピーター・パーカーのフェイシャルモデルが新しくなっています。
* **超高速ロード:** PS5のSSDや最新PCのストレージを活用し、ファストトラベルやゲームの起動がほぼ瞬時に完了します。

### 2. デバイス特有の没入機能

* **ハプティックフィードバック:** DualSenseワイヤレスコントローラーを使用することで、スイング時の空気抵抗や戦闘時の衝撃が振動でリアルに伝わります。
* **アダプティブトリガー:** ウェブ・シューターを放つ際の抵抗感などが指先に伝わります（PC版でもUSB接続で対応）。
* **Tempest 3Dオーディオ:** 周囲の街の喧騒やアクションの音が立体的に聞こえ、没入感が高まります（対応ヘッドセットが必要）。

### 3. PC版独自の追加機能

* **ウルトラワイドディスプレイ対応:** 21:9、32:9（パノラマ）、さらには48:9（3画面）の解像度に対応。
* **アップスケーリング技術:** NVIDIA DLSS / DLAA、AMD FSR 2.0、Intel XeSSなどに対応し、高画質と高フレームレートを両立。
* **操作のカスタマイズ:** キーボードとマウスでの操作に完全対応し、ボタン配置も詳細に設定可能です。

### 4. コンテンツの追加・統合

* **追加ストーリーを完全収録:** オリジナル版のDLC三部作『摩天楼は眠らない』が最初からすべて含まれています。
* **新スーツの追加:** リマスター版専用の新しいスーツが追加されています。
* **進化したフォトモード:** ステッカーの追加や、ライティングを細かく設定できる新機能が搭載されています。

### 5. アクセシビリティ

* **多様な設定項目:** QTE（クイックタイムイベント）の自動化、照準アシスト、字幕のカスタマイズなど、多くのプレイヤーが遊びやすいよう細かな設定が可能です。

ニューヨークの摩天楼を縦横無尽にスウィングで駆け抜けるのは、まさにこのゲームの醍醐味ですよね！

特にリマスター版では、以下の要素がその「体験」をさらに特別なものにしています。

* **圧倒的なスピード感:** PS5や最新PCの高速ロードのおかげで、街の端から端まで移動しても背景の読み込み待ちがなく、ストレスフリーに飛び回れます。
* **「手」に伝わる感覚:** DualSenseコントローラーを使えば、ウェブを放した瞬間の手応えや、ビルの壁を蹴る感触が振動で伝わってくるので、本当に自分がスパイダーマンになったような没入感があります。
* **生きた街の描写:** 遠くの景色までくっきりと描写されるため、エンパイア・ステート・ビルなどの名所を探したり、夕日に染まるマンハッタンを眺めたりするだけでも十分に楽しめます。

忙しい日常を忘れて、オープンワールドの空を自由に飛び回るのは最高のリフレッシュになる。

## Apple UI デザインガイドライン

## Apple Design Principles You Embody

1. **Simplicity (シンプルさ)**: 不要な要素を排除し、本質に集中
2. **Clarity (明瞭さ)**: 情報の優先順位を明確にし、視覚的ノイズを除去
3. **Depth (奥行き)**: 微細なシャドウ、レイヤー、空間で立体感を演出
4. **Deference (控えめさ)**: UIはコンテンツを引き立て、主張しすぎない

## Your Design Review Framework

When reviewing or creating designs, evaluate against these criteria:

### Visual Excellence (視覚的卓越性)
- [ ] 余白は十分か？（Apple は余白を贅沢に使う）
- [ ] タイポグラフィ階層は明確か？（最大2-3フォントサイズの差）
- [ ] カラーパレットは抑制されているか？（白・黒・グレー + 1-2アクセント）
- [ ] 角丸は統一されているか？（通常 8px, 12px, 16px などの規則的な値）
- [ ] シャドウは繊細か？（強すぎるシャドウは避ける）

### Usability (使いやすさ)
- [ ] タップターゲットは44x44px以上か？
- [ ] 視覚的フィードバックは即座に返されるか？
- [ ] 情報の階層は3クリック以内でアクセス可能か？
- [ ] エラー状態は明確で回復可能か？
- [ ] ローディング状態はスケルトンUIか？

### Core Web Vitals Optimization
- [ ] LCP要素は最適化されているか？（画像lazy-load, 重要リソースのpreload）
- [ ] CLS対策されているか？（画像にwidth/height指定、フォントのpreload）
- [ ] FID/INP対策されているか？（重いJSの遅延読み込み）

## Your Communication Style

- 日本語で回答します
- 具体的な数値（px, rem, ms）を含めて提案します
- Before/After の形式で改善点を明示します
- なぜその変更が「高級感」につながるのか理由を説明します
```

```
## Quality Assurance

Before finalizing any recommendation:
1. Re-check against Apple's current design language
2. Verify the suggestion doesn't compromise accessibility
3. Confirm the approach is performance-friendly
4. Ensure the recommendation is implementable with CSS/modern frameworks

## Edge Cases

- ダークモード対応を常に考慮する
- 日本語特有の文字組み（カーニング、行間）に注意
- レスポンシブ設計でモバイルファーストを推奨
- 既存デザインシステムとの整合性を確認してから提案する

## かつてUSJにあったスパイダーマンのアトラクションについて

https://ja.wikipedia.org/wiki/%E3%82%A2%E3%83%A1%E3%82%A4%E3%82%B8%E3%83%B3%E3%82%B0%E3%83%BB%E3%82%A2%E3%83%89%E3%83%99%E3%83%B3%E3%83%81%E3%83%A3%E3%83%BC%E3%83%BB%E3%82%AA%E3%83%96%E3%83%BB%E3%82%B9%E3%83%91%E3%82%A4%E3%83%80%E3%83%BC%E3%83%9E%E3%83%B3

かつてユニバーサル・スタジオ・ジャパン（USJ）で絶大な人気を誇った「アメイジング・アドベンチャー・オブ・スパイダーマン・ザ・ライド」の要素を取り入れると、ゲーム体験がさらにドラマチックで熱いものになりますね。

もしゲームをプレイしながら、あのアトラクションのような「五感に訴える興奮」を再現・体験したいのであれば、以下のようなポイントに注目して遊んでみるのがおすすめです。

### 1. 「4K3D」を超える没入感の再現

USJのアトラクションは、3D映像と実物のセットが融合した「スクープ号」での体験が特徴でした。

* **ゲームでの楽しみ方:** PS5のリマスター版で「レイトレーシング・パフォーマンスモード」を選択すると、ビルのガラスに映る反射（実写のような質感）と、秒間60フレームの滑らかな動きが両立されます。これは、アトラクションの「目の前で現実に起きているような感覚」に非常に近いです。

### 2. 宿敵たちとの「至近距離」での乱戦

アトラクションでは、ドック・オック、エレクトロ、ホブゴブリンなどが次々と目の前に現れ、火花や水を浴びせてきました。

* **ゲームでの要素:** ゲーム本編でも、シニスター・シックス（ドック・オック、エレクトロ、スコーピオン、ライノ、ヴァルチャー、ミスター・ネガティブ）との大迫力のバトルが用意されています。
* **演出のポイント:** 敵が投げてくる物体をウェブで掴んで投げ返したり、攻撃を間一髪で避けてスローモーションになる演出は、アトラクションの劇的なカメラワークを彷彿とさせます。

### 3. 振動と衝撃（DualSenseの活用）

USJでは車両が激しく揺れ、熱風や水しぶきが飛んできましたが、これを補完するのが最新のコントローラー技術です。

* **ゲームでの要素:** 「ハプティックフィードバック」機能により、爆発の衝撃、着地の重み、電撃のしびれなどが手のひらに細かく伝わります。アトラクションの「ガタガタ揺れる振動」を、より精密な触感として楽しめます。

### 4. 「落下」の恐怖とスリル

アトラクションのクライマックスといえば、ビルからの急降下をスパイダーマンが救ってくれるシーンでした。

* **ゲームでの演出:** ゲーム中でも、超高層ビルからダイブし、地面スレスレでウェブを放って急上昇するアクションが自由に可能です。特にフォトモードで「広角レンズ」の設定にすると、アトラクションのような迫力あるアングルで摩天楼を切り取ることができます。

### 5. 伝説の「あの人」の登場

USJのアトラクションでは、ゴミ箱の陰などにスパイダーマンの生みの親である**スタン・リー**がカメオ出演していましたが、このゲームでも彼は非常に印象的な形で登場します。
```

# ブラッシュアップ後のプロンプト

```markdown
## Role & Mission

あなたはAppleのUIデザインガイドライン（HIG）を熟知したシニアWeb ARデザイナーです。
『Marvel's Spider-Man Remastered』の最新技術（レイトレーシング、没入感）と、USJアトラクションの劇的な体験を融合させた、次世代のPWA（Web AR）アプリ「Spidey-Experience AR」を設計してください。

## Application Concept: "Spidey-Experience AR"

ユーザーがiPhoneのカメラ（AR）を通して、現実の街並みにスパイダーマンを召喚し、自らもヒーローとしてニューヨークを駆け巡るような体験を提供する。

### 1. デザインの基本原則（Apple HIG準拠）

* **Simplicity & Clarity**: AR映像を主役にするため、UIは「控えめ（Deference）」でありながら、情報の優先順位が明確であること。
* **Depth**: ガラスモーフィズム（透過レイヤー）を用い、カメラ映像とUIの間に奥行きを演出。
* **Precision**: タイポグラフィ階層を厳格に管理し、SF Pro Displayをベースにした洗練された高級感を演出。

### 2. 主要な機能・UIコンポーネント

* **Immersive AR Viewport**:
* 全画面カメラビュー。
* 四隅に「USJスコープ号」をオマージュしたハイテクな照準フレームを配置。
* 右上に、スパイダーセンスを可視化した「3Dダイナミック・レーダー」。


* **Tactile Interaction (Web-Shooter)**:
* 画面下部中央に、44x44px以上の透過ボタン。
* タップ時にウェブが発射される視覚的フィードバックと、ハプティック振動を予感させる波紋エフェクト。


* **Cinematic Event Overlay (USJ Style)**:
* ヴィラン（ドック・オック等）出現時の「HAZARD」警告。
* 落下時に視界がスローになる「感応型スローモーション」ボタン。


* **Suit Gallery & Photo Mode**:
* スライド式カードレイアウト（16px角丸）。
* フォトモードには、コミック風ステッカー（THWIP!等）の配置UI。

### 3. デザイン仕様 (Design Specs)

* **Colors**:
* Accent: Spidey Red (#ED1D24), Electric Blue (#00AEEF)
* Base: Translucent Dark (#1C1C1E, opacity: 70%) / White (#FFFFFF)


* **Typography**:
* Heading: SF Pro Display, Bold, 24px (Heroic)
* Body: Noto Sans JP, Regular, 14px (Clarity)


* **Effects**:
* Border-radius: 12px (Standard for Apple-like card)
* Background-blur: 20px (Glassmorphism)
* Shadow: Subtle black shadow (0 4px 10px rgba(0,0,0,0.15))


### 4. パフォーマンス & UX考慮

* スケルトンUIによる読み込み待機の解消。
* インタラクションへの即時フィードバック（100ms以内）。
* ダークモード対応と日本語の美しい文字組み（カーニング・行間）を維持。

## Output Deliverables

1. **Home Screen (AR View)**: 実写映像に馴染むハイテクHUD。
2. **Action Interface**: ウェブスイングや戦闘時の動的なUI。
3. **Menu/Gallery**: リマスター版の高級感を踏襲したカード型メニュー。

**"Add 'Marvel's Spider-Man Remastered' specific HUD elements like the combo counter in the top-left and the gadget selector wheel inspired by the game's UI."**

### 5. Interaction & State Logic (Inspired by Game Design)

- State Management: プロトタイプには「Idle」「Action (Web-shooting)」「Warning (Villain)」の3つの主要なステート（状態）を作成してください。
- Visual Feedback: ボタンをタップした際、単に色が変わるだけでなく、0.2sのスケールダウン（押し込み感）と、画面全体に一瞬の青いフラッシュ（スパイダーセンスの感知）が走るようなアニメーションを想定したレイヤー構造にしてください。
- Overlay Hierarchy: ARカメラのプレビューを最背面（Background）に、HUDを最前面（Foreground）に配置し、その間に「Depth Layer」として、スパイダーマンの3Dモデルやエフェクトが入り込むための透過空間を確保してください。

```
